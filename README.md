# Parcel experiments

Setting up React project with Parcel

*   See [Parcel js](https://parceljs.org/)

```bash
]$ npm install -g parcel-bundler
]$ yarn add react react-dom
]$ yarn add babel-preset-env babel-preset-react --dev
```

## Feature

*   React
*   ES6 with Babel
*   SCSS
*   CSS Modules
*   Typography.js
*   Build for production
